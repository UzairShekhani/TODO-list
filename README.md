## ✨ Features

- 📝 Add tasks to your list.
- ✅ Mark tasks as completed.
- 🗑️ Delete tasks from your list.
- 🌐 Responsive and visually appealing design.
- 📦 Automatically saves your tasks in the browser's local storage.
